📚 Online Book Store Management System (PostgreSQL)

Overview

The Online Book Store Management System is a database project developed using PostgreSQL. It is designed to manage the core operations of an online bookstore, including books, customers, authors, publishers, orders, and payments.

This project demonstrates the use of relational database concepts and SQL queries to efficiently store, organize, and retrieve data.

Features

- Manage books and categories
- Store customer information
- Track customer orders
- Maintain publisher and author records
- Record payment details
- Generate reports using SQL queries

Technologies Used

- PostgreSQL
- SQL

SQL Concepts Used

- CREATE TABLE
- INSERT, UPDATE, DELETE
- SELECT Queries
- JOINs
- Aggregate Functions
- GROUP BY & HAVING
- Subqueries
- Views
- Primary Key & Foreign Key
- Constraints

Project Structure

- "schema.sql" – Database schema
- "data.sql" – Sample data
- "queries.sql" – SQL queries
- "README.md" – Project documentation

Learning Outcomes

This project helped me strengthen my understanding of relational database design, SQL query writing, table relationships, data retrieval, and database management using PostgreSQL.
